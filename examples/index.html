This page is a simple example to showcase the Komponentor framework.

On top of the page you can see a navigation bar with links to the different views. When you click on a link, the komponentor router will load the corresponding komponent into the #app div. The views are actually komponents, each defined in their own file.

The separate komponents where CSS formated to be visually distinct from each other by featuring a dashed border and a box shadow.
